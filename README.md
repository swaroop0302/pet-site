# Pet Site 🐾

A full-stack **Pet Shop Management and E-Commerce Website** built using **PHP, HTML, CSS, JavaScript, and MySQL**. The project provides an online platform for customers to browse and purchase pet products while also giving administrators tools to manage products, suppliers, orders, stock, and customers.

## 🚀 Features

### 👤 User Features

* User registration and login system
* Browse pet products by category
* Add products to cart
* Online payment interface
* Order management
* Password reset and forgot password functionality
* Product categories for:

  * Dog food
  * Cat food
  * Fish food
  * Bird food
  * Pet medicines
  * Pet toys
  * Pet supplies

### 🛠️ Admin Features

* Secure admin login
* Add, update, and manage products
* Supplier management
* Customer management
* Order tracking and status updates
* Payment status management
* Stock availability management
* Dashboard metrics and reports

## ⚙️ Installation Guide

### 1️⃣ Clone Repository

```bash
git clone https://github.com/swaroop0302/pet-site.git
```

### 2️⃣ Move Project to WAMP Directory

Place the project folder inside:

```text
C:\wamp64\www\
```

### 3️⃣ Start WAMP Server

Start:

* Apache
* MySQL

### 4️⃣ Import Database

1. Open phpMyAdmin
2. Create a database
3. Import the SQL file (if available)

### 5️⃣ Run the Project

Open browser:

```text
http://localhost/project/
```
